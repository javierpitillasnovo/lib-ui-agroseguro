Código a incluir en TS para el correcto funcionamiento

Imports
import { StepTemplateDirective, UiStepperComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [StepTemplateDirective, UiStepperComponent]

Código de ejemplo HTML

<ui-stepper [steps]="steps" [buttonsDisabled]="false">
<ng-template stepContent [step]="1" let-item> Contenido a proyectar </ng-template>

</ui-stepper>

Propiedades input<>() del componente

[steps] : Input con el objeto de steps, número y título.
[buttonsDisabled] : Boolean para controlar si los botones del step en el que estamos están habilitados o no para pasar al paso anterior o siguiente.
[activeStep] : Step activo.
[showInitialButton] : Boolean para mostrar o no el botón inicial.
[showInitialButtonText] : String para definir el texto del botón inicial.
[showEndButton] : Boolean para mostrar o no el botón final.
[showEndButtonText] : String para definir el texto del botón final.

Propiedades output<>() del componente

[initialButtonClicked] : Evento que se emite cuando se hace clic en el botón inicial.
[endButtonClicked] : Evento que se emite cuando se hace clic en el botón final.

Proyecciones de contenido hacia el componente

Hace falta pasar tantas proyecciones como pasos tenga el step y es importante utilizar la directiva stepContent indicando el numero de step a continuación : [step]=1

<ng-template stepContent [step]="1" let-item> Contenido a proyectar </ng-template>
