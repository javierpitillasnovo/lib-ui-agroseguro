Código a incluir en TS para el correcto funcionamiento

Imports
import { UiLinkComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiLinkComponent]

Código de ejemplo HTML

<ui-link text="Action link" icon="icon-icon" additionalClasses="iFirst"></ui-link>

Links

Propiedades input<>() del componente

text : Texto del enlace
icon : Icono del enlace

additionalClasses :
secondary: Estilos de enlace secundario (color blanco)
iFirst: nos pone el icono a la izquierda del texto
