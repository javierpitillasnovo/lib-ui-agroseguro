Código a incluir en el servicio a cuestión donde queramos hacer llamadas HTTP para el correcto funcionamiento 

Imports
  import { CallApiService, RequestApiParams, RequestMethod } from '@agroseguro/lib-ui-agroseguro';

Dentro de la clase de nuestro servicio incluir:
  readonly #callApiService = inject(CallApiService);

Código de ejemplo llamada HTPP

    nombreFuncion(): Observable<any> {
        let requestParams: RequestApiParams = {
            method: RequestMethod.get, //Puede ser .get .post .put .delete
            url: `endpointURL`,
        };
        return this.#callApiService.callApi(requestParams);
    };

Ejemplo de suscripción en un componente

    readonly #servicioDeEjemplo = inject(ServicioDeEjemplo); //Aquí hay que inyectar el servicio a utilizar en cuestión

    let exampleSub = this.#servicioDeEjemplo.nombreFuncion().subscribe(
      (response:any) => console.log(response)
    );
