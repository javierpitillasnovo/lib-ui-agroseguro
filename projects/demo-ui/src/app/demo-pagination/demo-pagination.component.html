Código a incluir en TS para el correcto funcionamiento

Imports
import { UiPaginationComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiPaginationComponent]

Código de ejemplo HTML

<ui-pagination
[totalItems]="100"
[items]="10"
(paginationAction)="clg($event)"
>
</ui-pagination>

Propiedades Input()

items : Number con los items actuales.
totalItems : Number con los items totales.

Propiedades Output()

(paginationAction): Evento que emite al hacer click en el icono de página siguiente o anterior.
