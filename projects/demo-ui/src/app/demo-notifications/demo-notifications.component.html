Código a incluir en TS para el correcto funcionamiento

Imports
import { NotificationService } from '@agroseguro/lib-ui-agroseguro';

Dentro de la clase del componente incluir:

readonly #viewContainerRef = inject(ViewContainerRef);
readonly #notificationService = inject(NotificationService);

showNotification() {
  this.#notificationService.showNotification(this.#viewContainerRef, { message: 'Operación
  cancelada', hasError: true, });
}

Propiedades input<>() del componente

message : texto de la notificacion
hasError : True si es un error , false si no lo es
autoClear : Boolean para eliminar la notificación de forma automatica tras 3 segundos
