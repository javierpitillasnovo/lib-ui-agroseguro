Código a incluir en el servicio a cuestión donde queramos hacer uso del storage

Imports
  import { SessionService } from '@agroseguro/lib-ui-agroseguro';

Dentro de la clase de nuestro servicio incluir:
  readonly #sessionService = inject(SessionService);

Ejemplo de uso

    Guardar en sesión: this.#sessionService.set('nombreGuardadoEnSession', valor);
    Recuperar valor de la sesión: this.#sessionService.get('nombreGuardadoEnSession');
    Eliminar de sesión: this.#sessionService.remove'nombreGuardadoEnSession');
