Código a incluir en TS para el correcto funcionamiento

Imports
import { UiButtonComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiButtonComponent]

Código de ejemplo HTML

<ui-button text="Ejemplo Primary" icon="icon-arrow-down" additionalClasses="btn__primary xl iFirst" [disabled]="false" ></ui-button>

Propiedades input() del componente

text : Texto del botón.
icon : Icono a mostrar.
[disabled] : Boolean para deshabilitar el botón.
additionalClasses :
  btn__primary : Estilos de botón primario, fondo verde.
  btn__primary--red : Estilos de botón primario, fondo rojo.
  btn_secondary : Estilos de botón secundario, fondo blanco con borde verde.
  xl : Tamaño del botón.
  iFirst : Estilos para situar el icono a la izquierda.
