Código a incluir en TS para el correcto funcionamiento

Imports
import { UiAccordionComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiAccordionComponent]

Código de ejemplo HTML

<ui-accordion title="Test" [status]="0">
<div>contenido1</div>
</ui-accordion>

Tipos de acordeones:

contenido1
contenido2
contenido3

Propiedades input() del componente

title : Título del accordeón
[showIcon] : Boolean para mostrar o no el icono
[status] : Input de tipo number para el control del estado del accordeón
  0 === Verde,
  1 === Amarillo
  2 === Rojo.
[open] : Input de tipo boolean para controlar si el accordeón debe de estar abierto o no, por defecto en false.
