Código a incluir en TS para el correcto funcionamiento

Imports
import: { UiTextareaComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiTextareaComponent]

Código de ejemplo HTML

<ui-textarea
[required]="true"
label="Campo de texto normal"
placeholder="PlaceholderText"
helpText="Texto de ayuda de un campo normal"
icon="icon-icon"
additionalClasses="iFirst">
</ui-textarea>

** Para ver el funcionamiento completo, ver ejemplo de formulario reactivo en Inputs

Propiedades input<>() del componente

placeholder : Placeholder del campo
label : Etiqueta del campo
helpText : Texto de ayuda del campo
[required] : Valor booleano para indicar si es requerido o no
[disabled] : Valor booleano para disabled
[align] : Propiedad para elegir la alineación enum 'column' | 'row' por defecto a column
[width] : Ancho en píxeles, por defecto a 100%
[error]: Propiedad para controlar el estado de error
errorMessage: Texto de error a mostrar

Propiedades output<>() del componente

textAreaChanges : Evento que emite cuando hay cambios en el inputs
