Código a incluir en TS para el correcto funcionamiento

Imports
import { UiHeaderAppsComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiHeaderAppsComponent]

<ui-header-apps appName="Nombre de la aplicación">
<img src="../../../assets/img/logo-transparente.svg" alt="Logo" />
</ui-header-apps>

appName : Nombre de la aplicación a mostrar en el header
