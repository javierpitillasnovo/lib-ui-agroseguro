Código a incluir en TS para el correcto funcionamiento

Imports
import { UiTagComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiTagComponent]

Código de ejemplo HTML

<ui-tag type="check" [icon]="true" text="Lorem ipsum"></ui-tag>

Propiedades Input()

[status] : Input, enum con los siguientes valores : '0 | 1 | 2'

[icon] : Boolean para controlar si se muestra el icono o no

text : Texto de la etiqueta
