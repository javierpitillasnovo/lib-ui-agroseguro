Código a incluir en TS para el correcto funcionamiento

Imports
import { SessionService, UiHeaderComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiHeaderComponent]

Dentro de la clase del componente incluir:
readonly #sessionService = inject(SessionService);
constructor() {
  this.#sessionService.set('user', 'userTest');
}

Código de ejemplo HTML

<ui-header>
<img src="../../../assets/img/logo-transparente.svg" alt="Logo" />
</ui-header>

Propiedades input<>() del componente

Este componente no recibe ninguna propiedad input<>(), en cambio si que proyecta el contenido con un <ng-content></ng-content> lo que significa que severá todo lo que pasemos dentro. Esto debería de utlizarse solo y únicamente para pasar el logo deseado
