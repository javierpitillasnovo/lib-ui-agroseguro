Código a incluir en TS para el correcto funcionamiento

Imports
import { UiModalComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiModalComponent]

Dentro de la clase del componente incluir:
readonly #viewContainerRef = inject(ViewContainerRef);
readonly #modalService = inject(ModalService);

this.#modalService.open(DemoModalComponent, {size: 'l' }).subscribe({
  complete: () => {
    Nos suscribimos si queremos hacer algo en base a una acción que ocurre dentro de la modal
  },
});

Modal

Propiedades input<>() del componente

size : Propiedad para controlar de que tamaño queremos la modal (enum) <'xl' |'l'>
