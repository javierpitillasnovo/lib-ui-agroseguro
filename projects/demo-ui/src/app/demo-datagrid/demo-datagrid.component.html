Código a incluir en TS para el correcto funcionamiento

Imports
import { UiDatagridComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiDatagridComponent]

Código de ejemplo HTML

<ui-datagrid
filterId="id_ratio"
[headers]="['Desde', 'Hasta', 'Límite máximo de daños']"
[dataOriginal]="ratios()"
[fieldsToShow]="[
{ name: 'ratioDesde' },
{ name: 'ratioHasta' },
{ name: 'limiteMaximoDanios' }
]"
(itemAdd)="itemAdd('ratios')"
(itemSave)="itemSave($event, 'ratios')"
(itemDelete)="itemDelete($event, 'ratios')"
>
</ui-datagrid>

Propiedades Input()

title : Input Título de la tabla
filterId: En caso de querer usar alguna propiedad de la respuesta como id, sino se usa la propiedad id y si no la trae genera uno aleatorio.
[headers] * : Headers de la tabla, string[].
[dataOriginal] *: Data de la tabla, generalmente la recuperaremos de un endpoint.
[newItem] *: Nueva instancia de item del mismo tipo que la dataOriginal para generar uno nuevo con el botón de añadir.
[fieldsToShow] *:
  export interface fieldsToShow {
    name: string;   -->  Propiedad del objeto dataOriginal a mostrar.
    options?: any[]; --> Opciones del select, lo ideal es pasar una signal() computada que ya tenga los valores.
    selectedOption?: any;  -->  Propiedad del objeto correspondiente al valor del select.
    readonly?: boolean;  --> Propiedad que marca el campo como deshabilitado en el modo de edición.
  }
showAddButton : Input para ver o no el botón de añadir registros.
showPagination : Input para ver si queremos mostrar o no la paginación.

Propiedades Output()

  (itemSelected): Evento que emite el registro seleccionado al hacer click en los checkbox de la izquierda de la tabla.
  (itemAdd): Evento que emite un boolean cuando se añade un registro a la tabla.
  (itemSave): Evento que emite el registro guardado o editado al hacer click en el icono de guardar.
  (itemDelete): Evento que emite el registro eliminado al hacer click en el icono de eliminar.
  (anySelectChange): Evento que emite el valor seleccionado y nombre del select que cambia cuando se cambia una opción en cualquier select.
