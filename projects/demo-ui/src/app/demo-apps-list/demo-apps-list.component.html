Código a incluir en TS para el correcto funcionamiento

Imports
import { UiAccordionComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [UiAccordionComponent]

Código de ejemplo HTML

<ui-apps-list title="Favoritos" [appItems]="appsFavs()"></ui-apps-list>

Tipos de Listas:

Favoritos: Para mostrar varios ítems sin agruparlos, pasamos el input [appItems].
<ui-apps-list title="Favoritos" [appItems]="appsFavs()"></ui-apps-list>

Apps: Para mostrar grupos de apps con ítems, pasamos el input [appList].
<ui-apps-list title="Apps" [appList]="appsList()"></ui-apps-list>

Propiedades input() del componente

title : Título de la sección
[appList] : Lista de grupos de apps
[appItems] : Lista de apps

Propiedades output() del componente

  (favorite) : Output que emite al hacer click en la estrella de favoritos en un appItem
