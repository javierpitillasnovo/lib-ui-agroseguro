Código a incluir en TS para el correcto funcionamiento

Imports
import { TabTemplateDirective, UiTabsComponent } from '@agroseguro/lib-ui-agroseguro';

Dentro del decorador @Component incluir:
imports: [TabTemplateDirective, UiTabsComponent]

Dentro de la clase del componente incluir:

public tabData = ['Tab1', 'Tab2', 'Tab3'];

Código de ejemplo HTML

<ui-tabs [data]="tabData">
<ng-template tabContent title="Tab1" let-item> Contenido del tab1 </ng-template>
<ng-template tabContent title="Tab2" let-item> Contenido del tab2 </ng-template>
<ng-template tabContent title="Tab3" let-item> Contenido del tab3 </ng-template>
</ui-tabs>

Propiedades input<>() del componente

[data] : Propiedad con las cabeceras de las tabs

Proyecciones de la estructura del modal

Las tabs recibe directamente el contenido proyectado:

<ng-template tabContent title="Tab1" let-item> Contenido del tab1 </ng-template>
